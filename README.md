<h1 align="center">Jokenpô (Pedra, Papel e Tesoura)</h1>

<p align="center">
  Projeto desenvolvido com <strong>HTML, CSS e JavaScript</strong>, focado na prática de lógica de programação,
  manipulação do DOM e interação com o usuário.
</p>

<hr>

<h2>Sobre o Projeto</h2>

<p>
  Este projeto consiste em um jogo de Jokenpô no qual o jogador escolhe entre pedra, papel ou tesoura,
  enquanto a máquina realiza uma escolha aleatória. O resultado de cada rodada é exibido em tempo real,
  juntamente com um sistema de pontuação dinâmico.
</p>

<p>
  O objetivo principal foi consolidar conceitos fundamentais do front-end, utilizando JavaScript puro
  para controle de estado e atualização da interface.
</p>

<hr>

<h2>Funcionalidades</h2>

<ul>
  <li>Escolha interativa entre pedra, papel e tesoura</li>
  <li>Escolha aleatória da máquina</li>
  <li>Verificação de vitória, derrota ou empate</li>
  <li>Placar atualizado em tempo real</li>
  <li>Interface simples, responsiva e estilizada</li>
  <li>Feedback visual ao interagir com os botões</li>
</ul>

<hr>

<h2>Tecnologias Utilizadas</h2>

<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript (Vanilla JS)</li>
</ul>
